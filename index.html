      NEGARA OPEN SOURCE

      INDEKS DATA PUBLIK

      DOMPENG — NEGARA OPEN SOURCE

      NEGARA OPEN SOURCE — dashboard open source untuk ringkasan entitas dari dokumen publik internet, menghubungkan sinyal antar sumber, dan menampilkan sebaran kota secara agregat.

      Bukan institusi pemerintah; dashboard publik ini hanya menampilkan agregat dan sampel tersensor.

      • 757,265 entitas terindeks
      • 3,337 dokumen publik
      • 269 kota di peta
      • 90,783 entitas dengan lokasi kota

      Kota dengan entitas terbanyak

      1. Ibu Kota Nusantara — 14,815 entitas
      2. Pontianak — 12,809 entitas
      3. Sambas — 7,905 entitas
      4. Malang — 5,090 entitas
      5. Probolinggo — 5,000 entitas
      6. Pasuruan — 4,436 entitas
      7. Tapin — 4,210 entitas
      8. Lebak — 4,120 entitas

      Metodologi dan privasi

      Data diproses oleh pipeline summary, dipublikasikan sebagai agregat, dan disensor sebelum masuk dashboard publik. Peta memakai cluster kota; titik tidak memuat daftar individu.

      Glosarium

      Entitas berarti unit hasil ekstraksi; referensi silang berarti keterhubungan statistik antar field; indeks unik berarti field yang umumnya mengarah ke satu entitas.
